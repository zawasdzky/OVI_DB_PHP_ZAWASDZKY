OVI Conexión a base de datos con PHP

INTRODUCCIÓN

EL presente objeto virtual de aprendizaje muestra de manera sencilla la manera en que se puede conectar y administrar informacion pagina web cliente con un servidor que contiene una base de datos.

Para el tutorial se empleará un servidor web local Apache www.apache.org

Una Base de datos en MYSQL www.mysql.com

Un aplicativo MAMP gratuito para gestionar nuestro servidor local y nuestra base de datos

Crear base de datos en el servidor e ingresar los datos

Establecer conexion entre un sitio cliente y la base de datos en el servidor
