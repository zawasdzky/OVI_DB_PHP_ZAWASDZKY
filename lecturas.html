INSTALACION DE SERVIDOR LOCAL CON MAMP

QUE ES MAMP?

MAMP es un programa que instala un entorno de servidor local en cuestión de segundos en su computadora. Viene de forma gratuita, y se instala fácilmente. MAMP no comprometerá ninguna instalación existente de Apache que se esté ejecutando en su sistema. ¡Puede instalar Apache, PHP y MySQL sin iniciar un script o tener que cambiar ningún archivo de configuración! Además, si ya no se necesita MAMP, simplemente elimine la carpeta MAMP y todo vuelve a su estado original (es decir, MAMP no modifica el sistema "normal").



¿COMO INSTALAR MAMP?

Debe acceder a mamp.info descargar el archivo ejectable abrirlo e instalalo.

para iniciar el servidor debe dar clic en iniciar servidores.

Conexión con Base de datos

Archivo php de conexión

Debe crear un arcivo llamado conecion.php y alojarlo en la carpeta htdocs que se encuentra en la carpeta instalada MAMP. en sese archivo debe alojar el sguiente codigo que se muestra en el ejemplo:

Este simple ejemplo muestra como conectar, ejecutar una consulta, imprimir las filas resultantes y desconectar a una base de datos MySQL. Ejemplo #1 Ejemplo general de la extensión MySQL

<?php
// Conectando, seleccionando la base de datos
$link = mysql_connect('mysql_host', 'mysql_user', 'mysql_password')
    or die('No se pudo conectar: ' . mysql_error());
echo 'Connected successfully';
mysql_select_db('my_database') or die('No se pudo seleccionar la base de datos');

// Realizar una consulta MySQL
$query = 'SELECT * FROM my_table';
$result = mysql_query($query) or die('Consulta fallida: ' . mysql_error());

// Imprimir los resultados en HTML
echo "<table>\n";
while ($line = mysql_fetch_array($result, MYSQL_ASSOC)) {
    echo "\t<tr>\n";
    foreach ($line as $col_value) {
        echo "\t\t<td>$col_value</td>\n";
    }
    echo "\t</tr>\n";
}
echo "</table>\n";

// Liberar resultados
mysql_free_result($result);

// Cerrar la conexión
mysql_close($link);
?>

Insertar y borrar información en una base de datos

Ingresar datos en la base de datos

Para esto se debe crear un archivo llamado insertar.php el cual contiene el siguiente codigo

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "mydatabase";

// crear la conexión
$conn = new mysqli($servername, $username, $password, $dbname);
// revisar la coneccion
 if ($conn->connect_error) {
    die("eror de conexion: " . $conn->connect_error);
}
// insertando los datos en la base de datos:
$sql = "INSERT INTO mytable (firstname, lastname, email)
VALUES ('John', 'Doe', 'john@example.com')";

// si los tados se han insertado, entonces muestre el mensaje
 if ($conn->query($sql) === TRUE) {
    echo "datos insertados ";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>

Para borar un registro debe cambiar la consulta sql por "DELETE FROM" especificando cual es el registro id que desea borrar con el "WHERE"


$sql = "DELETE FROM mytabla WHERE id=3"

Actualizar un registro de la Base de datos

Actualizar registro

Para actualizar un registro debe especifirac cuales son los campos que desea cambiar y debe identificar cual es el identificador id del registro, para esto el archivo actualizar debe tener la sentencia UPDATE como en el siguiente ejemplo:

<?php
$servername = "localhost";
$username = "root";
$password = "";
$dbname = "mydatabase";

// crear la conexión
$conn = new mysqli($servername, $username, $password, $dbname);
// revisar la coneccion
 if ($conn->connect_error) {
    die("eror de conexion: " . $conn->connect_error);
}
// Actualizar el campo de un registro en la base de datos:
$sql = "UPDATE mutabla SET lastname='pedro' WHERE id=2";

// si los tados se han insertado, entonces muestre el mensaje
 if ($conn->query($sql) === TRUE) {
    echo "datos insertados ";
} else {
    echo "Error: " . $sql . "<br>" . $conn->error;
}

$conn->close();
?>
